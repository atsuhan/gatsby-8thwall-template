# atsuhan-gatsby-template

My template using Gatsby, react, typescript.

## Recomendation

- node.js  
   `version: 13.5.0`
- yarn  
   `version: 1.22.0`

## Get Started

1. install node modules

```
$ yarn
```

2. start development server

```
$ yarn start
```
